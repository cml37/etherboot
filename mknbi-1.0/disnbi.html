NAME

disnbi - display network bootable image

SYNOPSIS

disnbi [-e] nbifile

DESCRIPTION

disnbl is a program that to display in symbolic form the contents of a network bootable image created by mknbi.

-e Extract contents of image as well. The directory will be written to nbidir and the segments to segmentn where n is 0, 1, 2, etc.

BUGS
