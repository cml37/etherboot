1. Netbooting a 286 with a ramdisk

I have booted a 286 diskless, using a boot ROM on a network card, sending an image of a ramdisk containing a DOS startup floppy disk. The 286 then ran programs from the ramdisk. This could be used to send programs like telnet, print spoolers, routers and other standalone utilities that you want to run on a diskless (and maybe headless and keyless) 286.
